**Цель:** полностью освоить операции с векторами и матрицами, которые являются фундаментом для ML‑алгоритмов.

**Вводные данные:**  
Наборы численных данных (таблица с признаками), а также собственные примеры.

**Задача:** реализовать _с нуля_ (только NumPy — без sklearn‑solver‑ов):

1. **Базовые операции с векторами:**
    
    - норма (L1, L2);
        
    - скалярное произведение;
        
    - косинусная близость;
        
    - проекции векторов;
        
2. **Матричные операции:**
    
    - умножение матриц, транспонирование, обратная матрица;
        
    - детерминант, ранг, линейная независимость;
        
3. **Решение линейных систем:**
    
    - метод Гаусса;
        
    - метод LU‑разложения;
